UX/UI Design Bootcamp

Intro to industry and UX

This 9 week immersive course is catered to beginners with no previous design or technical experience. Students will be taught the fundamentals of User Centered Design and learn to validate ideas by going through the design process starting with User Research, Rapid Prototyping and ending with User Testing & Iteration. Students will then learn how to take their designs a step further by coding the prototype using front-end web development skills in HTML, CSS & JavaScript. The course will end with a capstone project where students will take a new product idea from validation to launch.

Temario del Curso

Módulo 1

Desarrollo Web
Front-End

Aprende las bases de la programación con JavaScript y el diseño de aplicaciones web dinámicas. Aprenderás a crear una experiencia de usuario atractiva y dinámica usando HTML, CSS y JavaScript.

  • Instalación del entorno de desarrollo
  • Introducción a la programación con JavaScript
  • Composición de páginas web con HTML5 y CSS3
  • Librerías JavaScript. jQuery
  • Colaboración de proyectos a través de Github

Módulo 2

MEAN Stack
Back-End

Aprende a crear aplicaciones apoyadas en bases de datos no relacionales usando Node.Js, ExpressJS y MongoDB. Estas son algunas de las tecnologías más demandada por las empresas.

  • Node.Js y programación asíncrona en JavaScript
  • Aplicaciones web usando el framework Express.js
  • Bases de datos no relacionales con MongoDB
  • Publica tus aplicaciones en el internet siguiendo buenas prácticas

Módulo 3

Microservicios
con Angular2

En este módulo aprenderás a crear tus propias APIs (interfaz de programación de aplicaciones) para hacer un diseño de microservicios. También aprenderás como crear aplicaciones usando Angular2.

  • Diseño de APIs con Node.JS
  • Diseño de aplicaciones MVC con Angular2
  • Creación de componentes web
  • Consumo de servicios web externos con MEAN Stack
